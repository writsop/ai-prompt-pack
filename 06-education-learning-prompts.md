# 🎓 Education & Learning Prompts (20 Prompts)

---

## Learning & Study

### 1. Learn Any Topic Fast
```
Teach me [TOPIC] from scratch.

My current level: [BEGINNER/INTERMEDIATE/ADVANCED]
My background: [RELEVANT KNOWLEDGE]
Learning style: [VISUAL/READING/HANDS-ON/EXAMPLES]
Time available: [HOURS PER WEEK]
Goal: [WHAT I WANT TO DO WITH THIS KNOWLEDGE]

Create a learning plan:
1. Prerequisite check (what do I need to know first)
2. Core concepts breakdown (most important 20% that covers 80%)
3. Explain each concept with:
   - Simple analogy
   - Real-world example
   - Common misconception
4. Practice exercises (progressive difficulty)
5. Resources ranked by quality (free first)
6. Milestones to track progress
7. Timeline to proficiency
```

### 2. Explain Like I'm Five
```
Explain [COMPLEX TOPIC] in simple terms.

Requirements:
- Use everyday analogies (no jargon)
- Build from what everyone knows
- Use a story or scenario if helpful
- Include a simple diagram described in words
- End with why it matters in daily life

Then provide three levels of explanation:
1. ELI5 (absolute beginner, use analogies)
2. ELI15 (high school level, introduce proper terms)
3. ELI-Expert (technical depth, assume foundations)
```

### 3. Feynman Technique Explainer
```
Help me use the Feynman Technique to deeply understand [TOPIC].

Step 1: I'll explain what I know:
[MY CURRENT UNDERSTANDING]

Please:
1. Identify gaps in my understanding
2. Point out where I used jargon without truly understanding it
3. Ask me clarifying questions that expose weak spots
4. Provide simple explanations for the gaps
5. Give me an analogy that captures the essence
6. Quiz me with 5 questions to test real understanding
```

### 4. Study Plan Creator
```
Create a study plan for [EXAM/CERTIFICATION/SKILL].

Exam/Goal: [NAME]

Current knowledge: [SELF-ASSESSMENT]
Hours available per week: [NUMBER]
```

### 5. Flashcard Generator
```
Create flashcards for [TOPIC/CHAPTER].

Source material: [PASTE OR DESCRIBE]
Number of cards: [20-50]
Difficulty: [BASIC RECALL/APPLICATION/ANALYSIS]

For each card:
Front: [Question or prompt]
Back: [Answer - concise but complete]

Card types to include:
- Definition cards
- Concept comparison cards
- Application scenario cards
- Fill-in-the-blank cards
- True/false with explanation cards

Organize by subtopic. Mark difficulty level (easy/medium/hard).
Format for easy import into Anki/Quizlet.
```

### 6. Book Summary and Action Items
```
Provide a comprehensive summary of [BOOK TITLE] by [AUTHOR].

Include:
1. One-sentence summary
2. Core thesis (3-4 sentences)
3. Key ideas (top 5-7 with explanations)
4. Most important quotes (5)
5. Chapter-by-chapter summary (2-3 sentences each)
6. Who should read this (and who shouldn't)
7. Action items: 5 specific things I can implement this week
8. Related books to read next
9. Criticisms and limitations of the book's arguments
```

## Teaching & Tutoring

### 7. Lesson Plan Builder
```
Create a lesson plan for teaching [TOPIC].

Students: [AGE/LEVEL]
Class size: [NUMBER]
Duration: [MINUTES]
Learning objectives: [WHAT STUDENTS SHOULD KNOW/DO AFTER]
Prerequisites: [WHAT THEY ALREADY KNOW]
Resources available: [TECH, MATERIALS, etc.]

Lesson plan:
1. Warm-up/hook (5 min) - grab attention
2. Direct instruction (15 min) - teach concepts
3. Guided practice (10 min) - do together
4. Independent practice (15 min) - students try
5. Check for understanding (5 min) - assessment
6. Wrap-up and preview (5 min)

Include:
- Differentiation strategies (advanced and struggling students)
- Engagement techniques
- Assessment rubric
- Homework/extension activity
```

### 8. Quiz Generator
```
Create a quiz on [TOPIC].

Level: [GRADE/DIFFICULTY]
Number of questions: [NUMBER]
Purpose: [FORMATIVE/SUMMATIVE]

Question types:
- 10 multiple choice (4 options each)
- 5 short answer
- 3 application/scenario-based
- 2 essay questions with rubric

For each question:
- The question
- Correct answer
- Explanation of why it's correct
- Common wrong answers and why students pick them
- Bloom's taxonomy level

Provide an answer key and grading guide.
```

### 9. Rubric Creator
```
Create an assessment rubric for [ASSIGNMENT].

Assignment: [DESCRIBE]
Level: [GRADE/COURSE]
Weight: [PERCENTAGE OF GRADE]

Criteria to assess:
1. [CRITERION 1]
2. [CRITERION 2]
3. [CRITERION 3]
4. [CRITERION 4]

For each criterion, define 4 performance levels:
- Exemplary (A) - exceeds expectations
- Proficient (B) - meets expectations
- Developing (C) - approaching expectations  
- Beginning (D/F) - below expectations

Include point values and descriptor for each cell.
Format as a table.
```

### 10. Student Feedback Writer
```
Write constructive feedback for a student's [ASSIGNMENT TYPE].

Student level: [GRADE/YEAR]
Assignment: [DESCRIBE]
Submission quality: [STRONG/AVERAGE/NEEDS WORK]
Specific issues: [LIST]
Strengths: [LIST]

Feedback structure:
1. Positive opening (specific praise, not generic)
2. Main strength with example from their work
3. Area for improvement #1 (with specific suggestion)
4. Area for improvement #2 (with specific suggestion)
5. Encouraging close with next steps

Tone: Supportive, growth-oriented. Use "I noticed" and "Consider trying" language.
```

## Research & Writing

### 11. Research Paper Outline
```
Create a research paper outline for [TOPIC].

Academic level: [UNDERGRADUATE/GRADUATE/DOCTORAL]
Field: [DISCIPLINE]
Required length: [PAGES/WORDS]
Citation style: [APA/MLA/CHICAGO]
Thesis statement: [YOUR ARGUMENT]

Outline:
1. Abstract (150-300 words)
2. Introduction (hook, background, thesis, roadmap)
3. Literature Review (organize by themes, not sources)
4. Methodology (if applicable)
5. Body sections (3-5 main arguments)
6. Counterarguments and rebuttals
7. Discussion
8. Conclusion (restate thesis, implications, future research)

For each section: key points, potential sources to find, approximate word count.
```

### 12. Thesis Statement Refiner
```
Help me refine my thesis statement.

Topic: [BROAD TOPIC]
My current thesis: [DRAFT THESIS]
Assignment type: [ARGUMENTATIVE/ANALYTICAL/EXPOSITORY]
Course: [SUBJECT]

Please:
1. Evaluate my current thesis (specific, arguable, supportable?)
2. Identify weaknesses
3. Provide 3 refined versions (from narrow to broad)
4. For each version:
   - Why it works
   - What evidence you'd need
   - Potential counterarguments
5. Recommend the strongest version and explain why
```

### 13. Citation Finder and Formatter
```
Help me find and format sources for my paper on [TOPIC].

Citation style: [APA 7TH/MLA 9TH/CHICAGO]
Sources needed: [NUMBER]
Types: [JOURNAL ARTICLES/BOOKS/WEBSITES]

For the topic, suggest:
1. 10 search terms for academic databases
2. Key journals in this field
3. Landmark studies to reference
4. How to evaluate source credibility
5. Citation format examples for each source type
6. In-text citation format
7. Reference list format
8. Common citation mistakes to avoid
```

### 14. Essay Feedback and Improvement
```
Review my essay and suggest improvements:

[PASTE ESSAY]

Assignment requirements: [DESCRIBE]
Grading criteria: [IF KNOWN]

Review for:
1. Thesis clarity and strength
2. Argument structure and logic
3. Evidence quality and integration
4. Paragraph organization (topic sentences, transitions)
5. Writing style (clarity, conciseness, voice)
6. Grammar and mechanics
7. Citation accuracy

For each issue: quote the problematic text, explain why, and rewrite it.
Overall grade estimate and what would elevate it to the next level.
```

## Language Learning

### 15. Language Practice Conversation
```
Have a conversation with me in [LANGUAGE].

My level: [A1-C2 / BEGINNER-ADVANCED]
Topic: [WHAT TO TALK ABOUT]
Focus area: [GRAMMAR/VOCABULARY/PRONUNCIATION]

Rules:
1. Speak to me in [LANGUAGE]
2. If I make a mistake, gently correct it with explanation
3. Introduce 3-5 new vocabulary words naturally
4. After each exchange, provide:
   - Correction (if needed)
   - New word with definition
   - Cultural note (if relevant)
5. Adjust difficulty based on my responses
6. At the end, summarize what I learned
```

### 16. Grammar Explainer
```
Explain this grammar concept in [LANGUAGE]: [CONCEPT]

My native language: [LANGUAGE]
My level: [LEVEL]

Explain:
1. The rule in simple terms
2. When and why to use it
3. Formation/pattern
4. 5 example sentences (simple to complex)
5. Common mistakes native speakers of [MY LANGUAGE] make
6. Comparison to similar concept in my native language
7. Exceptions to the rule
8. 5 practice sentences for me to try (with answers hidden)
```
