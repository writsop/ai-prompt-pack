# ✍️ Content Creation Prompts (40 Prompts)

---

## Blog Posts & Articles

### 1. SEO Blog Post Writer
```
Write a 1500-word SEO-optimized blog post about [TOPIC].

Target keyword: [PRIMARY KEYWORD]
Secondary keywords: [LIST 3-5]
Target audience: [WHO]
Tone: [professional/casual/authoritative]
Intent: [informational/commercial/transactional]

Structure:
- Hook opening (no generic intros)
- H2 and H3 subheadings incorporating keywords naturally
- Include a "Key Takeaways" box at the top
- Add internal linking suggestions: [YOUR SITE TOPICS]
- End with a clear CTA
- Meta title (under 60 chars) and meta description (under 155 chars)

Use short paragraphs (2-3 sentences max). Include at least one data point or statistic per section.
```

### 2. Listicle That Ranks
```
Create a comprehensive listicle: "[NUMBER] [ADJECTIVE] [TOPIC] for [AUDIENCE] in [YEAR]"

For each item include:
- Descriptive H3 heading
- 2-3 sentence explanation of why it's included
- One specific tip or example
- Pros and cons (brief)
- Who it's best for

Add an intro that hooks with a surprising stat and a comparison table summary at the end.
```

### 3. How-To Guide
```
Write a step-by-step how-to guide: "How to [ACHIEVE GOAL]"

Context:
- Reader's current level: [beginner/intermediate/advanced]
- Tools/resources needed: [LIST]
- Expected time to complete: [ESTIMATE]

For each step:
- Clear action-oriented heading
- What to do (specific instructions)
- Why it matters (brief)
- Common mistake to avoid
- Pro tip

Include a "What You'll Need" section at the start and "Troubleshooting" FAQ at the end.
```

### 4. Thought Leadership Article
```
Write a thought leadership article about [INDUSTRY TREND/OPINION].

My position: [YOUR STANCE]
Credentials: [WHY YOU'RE QUALIFIED]
Target publication: [LinkedIn/Medium/Industry Blog]

Structure:
- Contrarian hook that challenges conventional wisdom
- Personal anecdote or case study
- 3 supporting arguments with evidence
- Acknowledgment of counterarguments
- Forward-looking conclusion with prediction

Tone: Confident but not arrogant. Use "we" language to include the reader.
```

### 5. Case Study Writer
```
Write a compelling case study based on these details:

Client: [NAME/TYPE]
Industry: [SECTOR]
Challenge: [PROBLEM THEY FACED]
Solution: [WHAT YOU DID]
Results: [METRICS/OUTCOMES]
Timeline: [HOW LONG]

Format as:
1. Executive Summary (3 sentences)
2. The Challenge (with specific pain points)
3. The Solution (step-by-step approach)
4. Implementation (obstacles overcome)
5. Results (with before/after metrics)
6. Key Takeaways
7. Client Quote (generate a realistic one)
```

## Copywriting Frameworks

### 21. AIDA Copy Generator
```
Using the AIDA framework, write copy for [PRODUCT/SERVICE].

Product: [DESCRIPTION]
Audience: [WHO]
Medium: [EMAIL/LANDING PAGE/AD/SOCIAL]

Attention: Write 3 headline options that stop the scroll
Interest: Build curiosity with a surprising fact or story
Desire: Paint the after-state, use testimonials
Action: Create urgency with a specific CTA

Provide the full copy piece plus a breakdown of each AIDA section.
```

### 22. PAS Framework Writer
```
Using Problem-Agitate-Solution, write copy for [OFFER].

Problem: [WHAT PAIN DOES YOUR AUDIENCE HAVE]
Target: [WHO SPECIFICALLY]
Solution: [YOUR PRODUCT/SERVICE]

Problem section: Describe their current frustrating situation with specific details
Agitate: Make the problem feel urgent and costly (what happens if they don't fix it)
Solution: Introduce your solution as the obvious answer

Write this as a: [EMAIL / LANDING PAGE / AD / VIDEO SCRIPT]
```

### 23. Before-After-Bridge Copy
```
Create Before-After-Bridge copy for [PRODUCT].

Before: Describe the reader's current painful state
After: Paint a vivid picture of life with your solution
Bridge: Explain exactly how your product gets them there

Product: [DESCRIPTION]
Target customer: [WHO]
Key transformation: [FROM X TO Y]

Write 3 versions:
1. Short (social media post - 150 words)
2. Medium (email - 300 words)
3. Long (landing page section - 500 words)
```

### 24. Storytelling Copy
```
Write a story-driven sales page for [PRODUCT/SERVICE].

The story:
- Hero: [YOUR CUSTOMER - describe them]
- Villain: [THE PROBLEM/ENEMY]
- Guide: [YOUR BRAND - how you help]
- Plan: [3 SIMPLE STEPS]
- Success: [THE TRANSFORMATION]
- Failure: [WHAT HAPPENS WITHOUT YOU]

Use the StoryBrand framework. Include a hero's journey narrative that leads naturally to the offer. Add emotional triggers and specific details to make it real.
```

### 25. Objection-Handling Copy
```
Write copy that preemptively addresses the top objections for [PRODUCT].

Product: [DESCRIPTION]
Price: [AMOUNT]

Common objections:
1. "It's too expensive" 
2. "I don't have time"
3. "Will this work for me?"
4. "I've tried similar things before"
5. "I need to think about it"

For each objection, write:
- Acknowledgment (validate the concern)
- Reframe (shift the perspective)
- Proof (evidence it's not an issue)
- CTA (why acting now is best)

Tone: Empathetic, not pushy.
```

## Advanced Techniques

### 36. A/B Test Variant Generator
```
Generate A/B test variants for [ELEMENT].

Current version: [PASTE CURRENT COPY]
Element: [HEADLINE/CTA/EMAIL SUBJECT/AD COPY]
Goal: [INCREASE CLICKS/OPENS/CONVERSIONS]

Create 5 variants, each using a different approach:
1. Question-based
2. Number/statistic-based
3. Emotional trigger
4. Curiosity gap
5. Direct benefit

For each variant, explain the psychological principle behind it and predict which audience segment would respond best.
```

### 37. Content Gap Analyzer
```
Analyze content gaps in [NICHE/INDUSTRY].

My current content covers: [LIST TOPICS]
Top competitors: [LIST 3-5 WITH URLS]
Target audience: [WHO]

Identify:
1. Topics competitors cover that I don't
2. Questions my audience asks that nobody answers well
3. Emerging trends not yet saturated
4. Content format gaps (video, podcast, interactive)
5. Keyword opportunities (low competition, high intent)

Prioritize by: search volume estimate, competition level, alignment with my brand, effort to create.
```
