# 🎨 Personal & Creative Prompts (20 Prompts)

---

## Creative Writing

### 1. Story Starter
```
Help me write a [GENRE] story.

Premise: [ONE-SENTENCE IDEA]
Setting: [WHERE AND WHEN]
Main character: [BRIEF DESCRIPTION]
Tone: [DARK/HUMOROUS/WHIMSICAL/GRITTY/LITERARY]
Length: [SHORT STORY/NOVELLA CHAPTER/FLASH FICTION]

Provide:
1. Opening hook (first paragraph that grabs attention)
2. Character profile (motivation, flaw, desire, fear)
3. Plot outline (3-act structure with key beats)
4. Key scenes to write (5 pivotal moments)
5. Ending options (3 different endings to choose from)
6. Voice/style notes (sentence rhythm, POV, tense)
```

### 2. Character Developer
```
Create a deep character profile for [CHARACTER NAME/TYPE].

Role in story: [PROTAGONIST/ANTAGONIST/SUPPORTING]
Genre: [GENRE]

Develop:
1. Physical description (distinctive, not generic)
2. Background (childhood, education, formative events)
3. Personality (using specific traits, not adjectives)
4. Speech pattern (how they talk, catchphrases, verbal tics)
5. Motivation (what drives them, surface and deep)
6. Fatal flaw (what could be their undoing)
7. Secret (something no one else knows)
8. Relationships (how they interact with others)
9. Character arc (how they change through the story)
10. Interview: Write 5 questions and answers in their voice
```

### 3. Worldbuilding Assistant
```
Help me build a world for my [GENRE] story.

Core concept: [ONE-SENTENCE WORLD PREMISE]
Scale: [CITY/COUNTRY/PLANET/MULTIVERSE]
Tech/Magic level: [DESCRIBE]

Develop:
1. Geography and climate
2. Political structure
3. Economy (what people trade, currency)
4. Culture (customs, holidays, food, art)
5. History (3 major events that shaped this world)
6. Conflict (major tensions in this world)
7. Daily life (what's a normal day like for a regular person)
8. Rules (what's different from our world, and the consistent logic)
9. Sensory details (what does this world smell, sound, feel like)
10. Map description (key locations and their significance)
```

### 4. Dialogue Coach
```
Help me write better dialogue for this scene:

[DESCRIBE THE SCENE AND CHARACTERS]

Current dialogue attempt:
[PASTE YOUR DIALOGUE]

Improve by:
1. Making each character sound distinct
2. Adding subtext (what they mean vs what they say)
3. Using action beats instead of dialogue tags
4. Cutting unnecessary words (real people don't speak in complete sentences)
5. Adding tension or conflict
6. Showing character through speech patterns
7. Pacing (mix short and long exchanges)

Provide the rewritten scene with annotations explaining each change.
```

### 5. Poetry Generator
```
Write a poem about [TOPIC/THEME].

Style: [FREE VERSE/SONNET/HAIKU/LIMERICK/SPOKEN WORD]
Tone: [MELANCHOLY/JOYFUL/CONTEMPLATIVE/FIERCE]
Inspired by: [POET OR STYLE, if any]

Requirements:
- Use concrete imagery (no abstract cliches)
- Include sensory details (at least 3 senses)
- Employ [METAPHOR/SIMILE/PERSONIFICATION]
- If formal structure, follow the rules exactly

Also provide:
- Title options (3)
- Line-by-line annotation (for learning)
- Suggestions for performance/reading aloud
```
